# Lesson 3: Homework Assignment


## Shell Scripting Assignment: Website Availability Checker

---

### Requirements

1. **Script name:** `check_google.sh`

2. **Functionality:**
   - The script must print the **current date and time** at the start of the check.
   - It must check if `google.com` is reachable using:
     - **Ping (ICMP check)**
     - **HTTPS (HTTP status code response)**


3. **Commands to use:**
   - `date` → to get current date and time  
   - `ping` → to test network connectivity  
   - `curl` → to test HTTPS response  
   - `grep` → to filter text output  
   - Redirection (`>`, `>>`, `&> /dev/null`) as needed

4. **Script output format:**
   Example:

   ```
   ### Checking connectivity at: 2024-07-24 14:32:10
   - Ping check: google.com is reachable.
   - HTTPS check: google.com is responding.
   ```

5. **Bonus (optional, extra credit):**
   - Save each result (with date and time) into a log file named `google_check.log` in the user’s home directory.
   - Separate each log entry with a line of dashes.

6. **Execution:**
   - The script must be **executable** (`chmod +x check_google.sh`).
   - It must run correctly when executed as:
     ```bash
     ./check_google.sh
     ```
7. **cronjob setup**
    - set crontjob to run every 5 min
