## Services Overview

| Service | Port | Purpose |
|---------|------|---------|
| **Prometheus** | 9090 | Metrics collection, storage, and querying |
| **Grafana** | 3000 | Visualization dashboards and alerting |
| **cAdvisor** | 8080 | Container resource usage metrics |
| **Node Exporter** | 9100 | Host system metrics (CPU, memory, disk) |
| **Loki** | 3100 | Log aggregation and storage |
| **Promtail** | 9080 | Log collection agent |
| **Log Generator** | - | Sample application for testing logs |

---

## Setup Instructions

### 1. Navigate to Lab Directory
```bash
cd "labprep/prometheus-grafana"
```

### 2. Review Configuration Files
The following configuration files are pre-configured:
- `docker-compose.yml` - Complete monitoring stack
- `prometheus.yml` - Prometheus configuration
- `loki-config.yml` - Loki configuration 
- `promtail-config.yml` - Promtail configuration
- `grafana/provisioning/` - Grafana datasources and dashboards

### 3. Start the Monitoring Stack
```bash
# Start all services in detached mode
docker compose up -d

# Check service status
docker compose ps

# View logs (optional)
docker compose logs -f
```

### 4. Verify Services

After starting the stack, verify each service is accessible:

#### Prometheus
- **URL**: http://<ec2-ip>:9090
- **Test**: Go to Status вҶ’ Targets to see all monitoring targets
- **Expected**: All targets should be "UP" status

#### Grafana  
- **URL**: http://<ec2-ip>:3000
- **Credentials**: admin/admin (change on first login)
- **Test**: Check that Prometheus and Loki datasources are configured
- **Expected**: Both datasources should show green "connected" status

#### cAdvisor
- **URL**: http://<ec2-ip>:8080
- **Test**: View container metrics in the web interface
- **Expected**: See all running containers and their resource usage

#### Node Exporter
- **URL**: http://<ec2-ip>:9100/metrics
- **Test**: View raw metrics endpoint
- **Expected**: See system metrics in Prometheus format

#### Loki
- **URL**: http://<ec2-ip>:3100/ready
- **Test**: Check readiness endpoint
- **Expected**: Should return "ready"
