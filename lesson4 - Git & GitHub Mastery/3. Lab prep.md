# Lesson 4: Git & GitHub Mastery - Lab Preparation

## Pre-Lab Setup Requirements

### 1. Software Installation
Ensure all students have the following installed before the lab session:

#### Git Installation
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install git

# CentOS/RHEL
sudo yum install git

# macOS (using Homebrew)
brew install git

# Windows
# Download from https://git-scm.com/download/win
```
