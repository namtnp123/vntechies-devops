# Lesson 11: Jenkins Fundamentals & Pipeline Basics - Homework

## Assignment Overview
This homework assignment will reinforce your understanding of Jenkins pipelines by having you work with your own repositories and implement complete CI/CD workflows. You'll fork existing projects, customize pipelines, and demonstrate mastery of Jenkins fundamentals.

## Prerequisites
- Completed Practice Lab exercises
- GitHub account with organization or personal repositories
- DockerHub account for image registry
- Jenkins environment running and accessible

## Learning Objectives
By completing this homework, you will demonstrate ability to:
- Fork and manage Git repositories
- Customize Jenkins pipelines for personal projects
- Implement secure credential management
- Create end-to-end CI/CD workflows
- Apply Jenkins best practices in real scenarios

## Assignment 1: Repository Setup and Forking 

**Steps**:
1. **Fork the Frontend Repository**:
   - Go to: https://github.com/devopsway/devops-project-application-cowsay-frontend
   - Click "Fork" and fork to your GitHub account/organization
   - Clone your fork locally for inspection

2. **Fork the Backend Repository**:
   - Go to: https://github.com/devopsway/devops-project-application-cowsay-api
   - Click "Fork" and fork to your GitHub account/organization
   - Clone your fork locally for inspection
