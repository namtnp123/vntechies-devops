# Lesson 5: Docker Fundamentals - Practice Lab

## Lab Overview
**Duration**: 40 minutes  
**Objective**: Gain hands-on experience with Docker basics, container operations, and image management

---

## Exercise 1: First Steps with Docker (10 minutes)

### 1.1 Verify Docker Installation
```bash
# Check Docker version and info
docker --version
docker info

# View Docker help
docker --help
```


### 1.2 Run Your First Container
```bash
# Run the hello-world container
docker run hello-world

# Check what happened
docker ps -a
```


### 1.3 Explore Docker Images
```bash
# List downloaded images
docker images

# Get detailed information about hello-world image
docker inspect hello-world
```

**Tasks:**
- Record the image ID and size of hello-world
- Note the creation date and architecture

## Exercise 2: Working with Web Server Containers (10 minutes)

### 2.1 Run Nginx Web Server
```bash
# Run nginx in detached mode with port mapping
docker run -d -p 8080:80 --name my-nginx nginx

# Check running containers
docker ps

# Test the web server
curl http://localhost:8080
# Or open browser to http://localhost:8080
```

### 2.2 Container Interaction
```bash
# View container logs
docker logs my-nginx

# Execute commands inside the container
docker exec -it my-nginx bash

# Inside the container, explore the filesystem
ls /usr/share/nginx/html/
cat /usr/share/nginx/html/index.html
exit
```

### 2.3 Container Management
```bash
# Stop the container
docker stop my-nginx

# Start it again
docker start my-nginx

# Check container status
docker ps
```

## Exercise 3: Exploring Different Images (10 minutes)

### 3.1 Work with Ubuntu Container
```bash
# Run Ubuntu container interactively
docker run -it --name my-ubuntu ubuntu:22.04 bash

# Inside the container, explore the environment
whoami
cat /etc/os-release
ls /
apt update && apt install -y curl
curl --version
exit
```


### 3.2 Database Container
```bash
# Run MySQL container with environment variables
docker run -d \
  --name my-mysql \
  -e MYSQL_ROOT_PASSWORD=mypassword \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:8.0

# Wait for MySQL to start (about 30 seconds)
sleep 30

# Connect to MySQL
docker exec -it my-mysql mysql -u root -pmypassword -e "SHOW DATABASES;"
```

**Tasks:**
1. Compare the sizes of ubuntu, node, and mysql images
2. Note the different ways to pass configuration to containers
3. Observe how containers can expose different services

## Exercise 4: Image and Container Management (5 minutes)

### 4.1 Container Lifecycle Management
```bash
# List all containers (running and stopped)
docker ps -a

# Remove stopped containers
docker rm my-ubuntu

# Stop and remove running containers
docker stop my-nginx my-mysql
docker rm my-nginx my-mysql

# Verify cleanup
docker ps -a
```

### 4.2 Image Management
```bash
# List all images
docker images

# Remove specific images
docker rmi hello-world

# Pull different image variants
docker pull nginx:alpine
docker pull nginx:latest

# Compare image sizes
docker images nginx
```

### 4.3 Docker Hub Exploration
```bash
# Search for images on Docker Hub
docker search python

# Pull a specific Python image
docker pull python:3.9-slim

# Inspect the image
docker inspect python:3.9-slim
```

**Questions to Answer:**
1. What's the size difference between nginx:alpine and nginx:latest?
2. How many Python images are available on Docker Hub?
3. What information does `docker inspect` provide?

## Exercise 5: Basic Networking and Volumes (5 minutes)

### 5.1 Container Networking
```bash
# Run two containers and test connectivity
docker run -d --name web1 -p 8081:80 nginx:alpine
docker run -d --name web2 -p 8082:80 nginx:alpine

# Test both web servers
curl http://localhost:8081
curl http://localhost:8082

# Check container network information
docker inspect web1 | grep IPAddress
docker inspect web2 | grep IPAddress
```

### 5.2 Volume Mounting
```bash
# Create a local HTML file
mkdir -p ~/docker-lab/html
echo "<h1>Custom HTML Page</h1><p>Served from host filesystem</p>" > ~/docker-lab/html/index.html

# Run nginx with volume mount
docker run -d --name custom-web -p 8083:80 -v ~/docker-lab/html:/usr/share/nginx/html nginx:alpine

# Test the custom page
curl http://localhost:8083
```

### 5.3 Cleanup
```bash
# Stop and remove all containers
docker stop $(docker ps -q)
docker rm $(docker ps -aq)

# Remove unused images (optional)
docker image prune
```
